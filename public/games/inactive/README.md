### These games are no longer actively used!

Among other issues, they make requests directly to the server rather than via 
the `logEvent` JavaScript interface to the mobile app. They cannot be deployed 
until they are updated to conform to the latest app telemetry standards laid 
out in this document: 
https://docs.google.com/document/d/1vYOuuMzA0MMBlMFcGZezYffeANwB4KH6VhoTq8baFsY
