===============
Fruit Sorting Event List
===============
LEVEL_START => Player start the game
	- LEVEL_START (event)
---------------
TAP => Everytime player choose the fruit position
	- CORRECT_RESPONSE/INCORRECT_RESPONSE (event)
	- Apple/Banana/Pear/Orange (fruit)
---------------
LEVEL_FAILED => Player lose time end
	- LEVEL_FAILED (event)
	- 0 (correct)
	- 0 (incorrect)
	- 0 (accuracy)
---------------
LEVEL_RESTART => Player restart the game
	- LEVEL_RESTART (event)
