File structure

The file structure is very simple.

The assets folder holds game resources (graphics, sounds etc).
The js folder holds game code.
The index.html file is the starting point of the game.

Quickstart

Create a folder inside your site public directore. Paste all the files there. For example if you create a folder named "game" inside your public folder, you can play the game by going to http://yoursite.com/game.

Custom HTML element

If you wish to embed the game into a custom element on the existing page, do the following steps:

  1. Copy "dist" and "assets" folders into the same folder as your .html file
  2. Add <script src="dist/bundle.js"> line right before the closing head tag.
  3. Create an element and give it a "game" id. For example: <div id="game"></div>
  4. Enjoy the game

Reskinning the game

The reskinning process is very simple, don't be discouraged by the fact that you'll need to use a third-party program.
Follow these steps.

  1. Download and install Texture Packer
    2. It's a free program that will allow you to update the game's graphics.
  2. Run Texture packer
  3. Double click on atlas.tps file in the assets folder.
  4. Change any file you want and then press publish sprite sheet
Now you can change any image in the folder and Texture Packer will automatically pack everything into a nice texture atlas. Don't change file names and try to get as close to the initial image sizes as possible.

When you're finished, click on the Publish sprite sheet button. Name the export atlas and you will get 2 files: atlas.png and atlas.json. Put both of them into the assets/images folder of the game.
